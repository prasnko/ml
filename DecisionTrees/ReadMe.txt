
The code is written in Python 2.7 using Windows 7 x86 platform.

The training and test data is read in code from txt files and not dat files.

Steps to compile:

1. Place DecisionTree.py and both the training and test files in C:\Python27

2. type python DecisionTree.py train-file test-file
 
e.g. python DecisionTree.py train.txt test.txt
